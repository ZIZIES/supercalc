# supercalc

all-in-one calculator with basic, scientific, and programmer modes

## modes

### basic
standard calculator operations
- addition, subtraction, multiplication, division
- decimal support
- clear function

### scientific
advanced math functions
- trigonometry: sin, cos, tan
- logarithms: log, ln
- operations: square root (√), square (x²), power (^)
- constants: π
- utilities: reciprocal (1/x), sign change (±)

### programmer
base conversions and bitwise operations
- bases: decimal, hexadecimal, binary, octal
- live conversion between bases
- bitwise operations: AND, OR, XOR
- hex digit support (A-F)

## how to use

1. go to https://supercalc.vercel.app/
2. pick your mode at the top
3. do math
4. profit
